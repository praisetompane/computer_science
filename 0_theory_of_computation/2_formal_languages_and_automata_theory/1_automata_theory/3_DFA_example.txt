Test Cases (Neso Academy):

    1. input = '001' 
        DFA processing:
            0    0    1
        A -> B -> B -> B
        result:
            last state = B
            B = final state in definition
            B = B
            ∴ string is valid

    2. input = '101'
        DFA processing:
            1    0     1
        A -> C -> C -> C
        result:
            last state = C
            C ≠ final state in definition
            C ≠ B
            ∴ string is invalid
